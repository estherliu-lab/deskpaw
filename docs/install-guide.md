# Install Guide / 安装说明

## 中文

### 如何添加到手机主屏幕

在 Safari 或 Chrome 中打开 DeskPaw 网站，点击分享或浏览器菜单，选择“添加到主屏幕”。添加后可以像 app 一样打开宠物小屋。

### 如何使用 PWA

DeskPaw V1 会把宠物档案和图片默认保存在本地浏览器。离线时，已缓存页面可以继续打开。

### 如何下载桌面宠物

第一版暂不提供真实桌面宠物安装包。`apps/desktop-pet/` 中预留了 Tauri / Electron 路线说明。

### 手机端和电脑端区别

手机端是 PWA 宠物小屋；电脑端的最终目标是透明、可拖拽、可置顶的桌面宠物。

## English

### How to install on mobile home screen

Open DeskPaw in Safari or Chrome, use Share or the browser menu, then choose Add to Home Screen.

### How to use the PWA

DeskPaw V1 stores pet profiles and images locally in the browser by default. Cached pages can still open offline.

### How to download the desktop pet

V1 does not ship a real desktop package yet. `apps/desktop-pet/` documents the Tauri / Electron path.

### Difference between mobile and desktop versions

Mobile is a PWA pet home. Desktop is planned as a transparent draggable floating companion.
