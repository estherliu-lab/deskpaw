# Roadmap

## V1

- Runnable Vite + React PWA.
- Mock pet generation from a local photo.
- 17 styles and 15 actions.
- Pet home interactions.
- Canvas share card download.
- Local privacy-first storage.

## V1.5

- IndexedDB image storage.
- More share-card templates.
- Better crop controls.
- QR-code support for share cards.
- More nuanced motion states.

## V2

- Optional AI background removal and image generation.
- Tauri desktop shell.
- Transparent borderless pet window.
- Tray menu and always-on-top mode.
- Local notification reminders.

## V3

- Optional account and cloud sync.
- Multi-pet homes.
- Widget integrations.
- Community style/action packs.
